# Homework 8: Analysis and Application of Remote Sensing & Geospatial Information

## Overview

This project focuses on the analysis and application of remote sensing and geospatial information techniques, specifically using ARIA v5 data for the Mataian area.

## Files

- `ARIA_v5_mataian.ipynb` - Jupyter notebook containing the main analysis workflow for ARIA v5 data processing and visualization

## Requirements

### Python Libraries
- numpy
- pandas
- matplotlib
- geopandas
- rasterio
- xarray
- jupyter

### Installation
```bash
pip install numpy pandas matplotlib geopandas rasterio xarray jupyter
```

## Usage

1. Open the Jupyter notebook:
   ```bash
   jupyter notebook ARIA_v5_mataian.ipynb
   ```

2. Follow the notebook sections:
   - Data Loading and Preprocessing
   - Data Analysis
   - Results and Visualization
